Adidas US Sales Dashboard (Power BI)

 📌 Project Overview
This project presents an interactive Power BI dashboard built using Adidas US sales data. The dashboard provides insights into sales performance, profit trends, and regional analysis.

 🚀 Key Features
- Sales and profit analysis
- Region-wise and state-wise performance tracking
- Product category insights
- Retailer-wise comparison
- Interactive filters and visuals

 🛠 Tools Used
- Power BI

 📊 Insights
